Shared Infrastructure

Core Services

14 platform services shared by all business suites. No business service duplicates their functionality — it always delegates. This keeps the system clean, auditable, and consistent.

Architecture rule
Finance never sends an SMS. HR never writes its own file upload handler. Education never builds a PDF report renderer. Each business service calls the appropriate shared service. Violations create hidden duplication and break the audit trail.
9 Levels · 27 Roles

Roles & Access Control

Hierarchical RBAC with data scoping. Every user carries: Role + Permissions + Department + Data Scope. The API middleware auto-filters records — no manual scoping in business logic.
